розробити single-page application, SPA

1) index.html
показує користувачу форму з одним текстовим полем і кнопкою

2) користувач вводить число і натискає на кнопку

3) в бекграунді відправляється запит json на сервер з введеним числом
наприклад
{
value : '124'
}

4) сервер парсає json і конвертує 124 в число прописом в гривні повертає результат клієнту такоє json
{
result : 'сто двадцять чотири гривні'
}

5) на клієнті в  index.html має показати результат 'сто двадцять чотири гривні'

6) користувач може ввести інше число і знову відправити запит на сервер відповідно результат має оновитися

На серверній частині реалізувати компоненту для конвертації числа прописом
застосувати phpunit 

Обробити помилки на введення некоректного числа 
специфіка складності алгоритмічно

 ще є в мене додаткове завдання

використай лібу https://github.com/PHPOffice/PhpSpreadsheet

і реалізуй 

composer run-script generateExcell


там просто в composer.json треба додати


{
    "scripts": {    
        "generateExcell": [
            "php generateExcell.php"
        ]
    }
}

результатом має бути ексель файл з табличкою
де є дві колонки

1 | одна гривня
2 | дві гривні
....
999 | девятсот девяносто девять гривень

ну ти зрозумів треба заюзати твою компоненту
